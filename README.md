# whisperx-translate-nas

Batch transcription & translation tool powered by WhisperX.

Optimised for Apple Silicon (MPS) but works on CPU and CUDA as well.

## Setup (PDM)

1. Install [PDM](https://pdm.fming.dev):
   ```bash
   pip install pdm
   ```
2. Install dependencies:
   ```bash
   pdm install
   ```
